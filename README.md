## Installation


```bash
npm install -g vuepress
```

## Launch

### development
```bash
vuepress dev
```

# production
```bash
vuepress build
```

## Changing the documentation

Directory structure can be changed in `.vuepress/config.js`. Every file needs to be addedd manually to the config file in the right parent directory.
